# Heritage Storefront - AI Context

## Project Overview
**Lead-gen site** for luxury jewelry specializing in custom black spinel engagement rings and heritage pieces crafted by multigenerational artisans. Dark academia aesthetic meets Souls-like minimalism.

**Current Stage**: Lead generation (consultation bookings)  
**Future**: E-commerce with product pages, collections, checkout  
**Primary CTA**: "Book a Free Consultation"

**Stack**: Next.js 15.5.10 (Pages Router, static export) + Tailwind CSS 4.1.18 + Framer Motion 12.29.2 + Radix UI  
**Deployment**: Netlify (static hosting)  
**Forms**: React Hook Form + Zod → Netlify Forms

## Core Principles

### 1. Design Philosophy - "Souls-like Gallery Minimalism"
- **Cold, not warm**: Pure blacks (#000000), desaturated greys, metallic accents
- **Minimal, not ornate**: Museum-like spacing, subtle textures, restrained animation
- **Sophisticated brutalism**: Inspired by Elden Ring, Metropolitan Museum, dark metal aesthetic
- **Sharp edges**: Zero border radius, angular corners, no roundness
- **Performance first**: Lightweight textures, optimized animations, no background-attachment: fixed

### 2. Writing Tone & Voice
**Primary inspiration**: https://royalkarkhana.com/

**Tone characteristics**:
- Sophisticated and reverent, yet accessible
- Educational without condescension
- Passionate about craft, materials, and heritage
- Aspirational (heirlooms, not accessories)

**Literary influences** - Blend these voices:
- **Tom Robbins' *Jitterbug Perfume***: Playful erudition, unexpected connections between ancient wisdom and material craft, sensory-rich descriptions, philosophical tangents that illuminate rather than distract
- **Maria Popova's *The Marginalian* (Brain Pickings)**: Intellectual curiosity meets poetic synthesis, deep research worn lightly, finding universal meaning in specific details, generous citation of sources and lineage
- **Patrick Rothfuss' *The Kingkiller Chronicle***: Precise technical language about craft (sympathy/naming = metalworking/gemology), reverence for mastery, the romance of difficult skills patiently learned, folklore embedded in material reality
- **Elden Ring/Dark Souls/D&D/Skyrim**: Cryptic item descriptions that imply vast histories, flavor text that rewards close reading, mysterious fragments that suggest deeper lore, archaic phrasing without pretension

**How these blend**:
- **Robbins' whimsy** tempers **Souls' darkness** → Mysterious but not oppressive
- **Popova's research depth** grounds **Rothfuss' romanticism** → Credible expertise
- **RPG terseness** structures **Robbins' digressions** → Focused but layered
- **Rothfuss' craft detail** meets **Popova's interdisciplinary reach** → Gemology connects to mythology, metallurgy to philosophy

**Key vocabulary**:
- Use: "multigenerational artisans," "ethically sourced," "handforged," "heritage pieces", "jewelsmithing", "gemologist", "mineralogy"
- Use: "connoisseurs," "authenticity," "lineage," "traditional methods"
- Use: "provenance," "temperament" (of metals), "inclusions" (not flaws), "facets" (literal + metaphorical)
- Avoid: "high-quality," "premium," "luxury" (unless paired with specifics)
- Avoid: Corporate marketing speak, hype, superlatives

**Writing techniques**:
- **Start with the material**: "Black spinel cleaves along octahedral planes" (specific) before "represents protection" (symbolic)
- **Embed history in description**: "The same bezel-setting method that secured Mughal court jewels now cradles this 2.5ct stone"
- **Let items "speak"**: Channel Souls item descriptions → "Ring. Bezel-set spinel, 18kt gold. Forged in the old way. The stone remembers darkness."
- **Draw unexpected connections**: Robbins-style → "Beet extract once dyed these silks the color of temple offerings; now, turmeric and madder root perform the same alchemy your grandmother's hands knew"
- **Write to show**: Write to teach, to inform, to enlighten, to inspire. Not to sell. Not to codemn competetors. Craft is a language that speaks to the soul. It's a way to tell stories that transcend time and space.
- **Cite lineage like Popova**: "As Ananda Coomaraswamy wrote in *The Transformation of Nature in Art*, traditional craft is 'not self-expression but surrender to the logic of materials'—a philosophy this workshop has embodied for 14 generations"

**Example good copy** (blending all influences):
> For seven generations, artisans in India and Thailand have been passing down the knowledge of jewelcrafting. Harder than most gemstones. And far more durable than diamonds for daily wear. 

> Each piece is hand-forged using techniques refined across centuries—methods that prioritize durability over speed, integrity over volume. Above all, we prioritize the quality of the materials and craftsmanship.

> From first conversation to finished piece: how a commission moves through sketch, stone selection, metalwork, and setting. The process takes weeks because each step requires decisions that can't be rushed—the hand that wears the ring must shape its making.

**Example bad copy**:
> "Our premium luxury rings are made with high-quality materials by expert craftsmen. Shop now for the best deals!"

**Practical application by content type**:
- **Product descriptions**: Souls-terse + Rothfuss-technical (materials, methods, measurements first; symbolism earned, not assumed)
- **Blog posts**: Popova-deep + Robbins-playful (research-driven but essayistic, historical tangents that illuminate craft)
- **About page**: Rothfuss-romantic + Popova-cited (tell the artisan story with specifics, name the lineage, credit the tradition)
- **Landing page**: RPG-cryptic + Marginalian-poetic (intrigue before exposition, questions before answers)

## Development Workflow

### Common Commands
```bash
npm run dev      # Development server (localhost:3000)
npm run build    # Static site generation → /out directory
npm run start    # Serve static build locally
npm run lint     # ESLint check
```

### Code Style
- **Components**: PascalCase files (`HeroSection.jsx`, `Header.jsx`)
- **Utilities**: camelCase files (`utils.js`, `products.js`)
- **Naming**: Descriptive, full words (avoid abbreviations)
- **Functional components**: Always use hooks, never class components
- **Imports**: Absolute paths with `@/` alias

### Git Conventions
- **Commit format**: Conventional Commits (`feat:`, `fix:`, `docs:`, `style:`, `refactor:`)
- **Example**: `feat: add multi-step consultation form`, `fix: mobile menu scroll lock`

## Key Technical Details

### Tailwind CSS v4 Migration (Completed 2026-01-27)
- **Config location**: CSS-first configuration in `styles/globals.css` using `@theme` block
- **No tailwind.config.js**: Removed (Tailwind v4 uses CSS variables instead)
- **PostCSS**: Uses `@tailwindcss/postcss` plugin
- **Import**: Changed from `@tailwind base/components/utilities` to `@import "tailwindcss"`

### Next.js Configuration
- **Static export** enabled (`output: "export"`)
- **Trailing slashes** added (`trailingSlash: true`)
- **Image optimization** disabled (`unoptimized: true`) for static export

### State Management
- **Cart**: React Context API (`CartContext.jsx`) with localStorage persistence
- **Forms**: React Hook Form + Zod validation
- **Page state**: Local `useState` hooks (no Redux/Zustand)

### Forms Submission
- **Netlify Forms** endpoint: POST to `/` with `form-name: "consultation"`
- **Validation**: Zod schemas before submission
- **Success redirect**: `/consultation/success`

### Performance Optimizations Applied
- Removed `console.error()` from production code
- No `background-attachment: fixed` (causes scroll jank)
- Lightweight SVG textures with low opacity (0.05)
- Fog vignette as fixed ::before pseudo-element
- Optimized grain texture (baseFrequency 0.9, numOctaves 3)

## Critical Guidelines

### DO ✓
- Follow Royal Karkhana's writing style (sophisticated, reverent, educational)
- Use colors from `design.md` (void, stone-dark, pale-gold, silver-white)
- Use `font-display` (Cinzel) for headings, `font-body` (Crimson Text) for paragraphs
- Keep animations snappy (200-500ms max, not 1000ms+)
- Validate with Zod before form submission
- Use `cn()` utility for conditional classnames
- Test mobile responsiveness (min 320px width)
- Use `gap-space-*` on containers instead of margins on children
- Write semantic HTML with ARIA labels

### DON'T ✗
- Use deprecated colors (ink-black, parchment, aged-gold, bone-white, ash-grey, etc.)
- Write warm/ornate copy (no "premium luxury," "amazing quality")
- Create slow animations (>800ms duration)
- Add rounded corners (`rounded-lg`) - design uses sharp edges
- Stack multiple SVG textures (performance issue)
- Use `background-attachment: fixed` (scroll jank)
- Add `console.log()` or `console.error()` to production code
- Guess Netlify Forms structure (check existing `consultation.jsx`)
- Add emojis or casual language
- Create new files without reading existing patterns first

### When Making Changes
1. **Read existing files first** - Match patterns exactly
2. **Check design.md** - Colors, fonts, component classes
3. **Test forms** - Validate Netlify Forms endpoint syntax
4. **Check mobile** - All changes must work 320px-1920px
5. **Run `npm run build`** - Ensure static export succeeds
6. **No console statements** - Remove all console.log/error before committing
