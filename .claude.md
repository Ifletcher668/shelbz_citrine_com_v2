# `.claude.md` - Heritage Jewelry Enterprise Website Specification

## Project Overview

**Heritage Jewelry Enterprise** is a bespoke jewelry consultation platform that bridges ethical Indian craftsmanship with discerning US clients. We offer two core services:

1. **Custom Jewelry Consultation**: Black spinel engagement rings and heritage pieces crafted by Royal Karkhana artisans in India
2. **Curated Mystery Boxes**: Monthly subscription boxes of authentic Indian goods (textiles, art supplies, beauty products, etc.)

This website serves as the primary funnel for consultation bookings and mystery box subscriptions while educating visitors about authentic craftsmanship, ethical sourcing, and traditional techniques.

---

## Mission Statement

We bridge the gap between American demand for high-quality, natural, ethical goods and the lack of accessible supply. In a world of cheap plastic garbage, we educate people on what is REAL—connecting them directly to multigenerational artisans who create heirloom pieces using traditional methods.

**Tone**: We inform, enlighten, and elucidate—not sell. This is a digital manuscript, not a corporate landing page. We educate about craft without disparaging others' choices.

## Technical Stack (Free-Tier Optimized)

### Hosting & Deployment
- **Netlify** (free tier: 100GB bandwidth, 300 build minutes/month)
  - Static site generation for maximum speed
  - Automatic HTTPS
  - Form handling (100 submissions/month free)
  - Deploy previews for testing

### Framework
- **Next.js 15** with Static Site Generation (SSG)
  - Pre-render all pages at build time for blazing speed
  - Export as static HTML/CSS/JS
  - Excellent SEO out of the box
  - Easy migration path to e-commerce (Next.js supports dynamic routes, API routes, etc.)

### Content Management
- **Contentful** (free tier: 25K records, 2 locales, 5 users)
  - Headless CMS for blog posts, gem gallery, process documentation
  - Rich text editor for blog content
  - Asset management for images
  - **Migration strategy**: Content is accessed via API, making it straightforward to swap CMS backends later. Consider Sanity.io, Strapi (self-hosted), or Payload CMS as alternatives if Contentful becomes limiting.

### Forms & Data Collection
- **Netlify Forms** (free tier: 100 submissions/month)
  - Consultation requests
  - Mystery box questionnaires
  - Contact forms
- **Alternative if volume exceeds**: Formspree (50 submissions/month free) or Google Forms integration

### Email Notifications
- **Netlify Form Notifications** → your email
- Future: **SendGrid** (100 emails/day free) or **Resend** (3K emails/month free) for automated confirmations

### Styling
- **Tailwind CSS** with extensive custom configuration
- **Framer Motion** for slow, deliberate animations
- Design system: Dark Academia + Fantasy (D&D/Video Games) + Mass Accessibility

### Analytics
- **Plausible** (open-source, privacy-focused) OR
- **Google Analytics 4** (free, more robust)

### Future E-commerce Preparation
- **Stripe** for payments (no monthly fee, 2.9% + 30¢ per transaction)
- Next.js architecture supports easy addition of:
  - `/api` routes for Stripe webhooks
  - Dynamic product pages
  - Shopping cart context (already present in prototype)

## Design System - "Candlelit Study Meets Fantasy Craft"

### Design Philosophy

**What We ARE**:
- A digital manuscript / illuminated tome
- Dark academia aesthetic (aged parchment, candlelight, oxidized metals)
- Fantasy game UI (Guild Wars 2, Elden Ring, Elder Scrolls)
- Artisan portfolio (Devran Dogaroglu, Kate Vass Galerie)

**What We ARE NOT**:
- Corporate SaaS landing page
- Sterile minimalism
- Bright, cheerful e-commerce
- Fast-paced, "modern" UI

### Visual References
Study these sites for aesthetic direction:
- https://www.guildwars2.com/en/ (texture, ornamental borders)
- https://www.devrandogaroglu.com/ (asymmetric layouts, generous whitespace)
- https://www.katevassgalerie.com/ (serif-heavy, muted palette)
- https://www.mattleblancart.com/ (craft-focused, slow animations)
- https://en.bandainamcoent.eu/elden-ring/elden-ring (weathered metal, fantasy typography)
- https://elderscrolls.bethesda.net/en (old-world fonts, parchment textures)

## Layout Philosophy - "Broken Grid"

### Spatial System (Golden Ratio, Not 8px Grid)

```css
/* Spacing Scale - Based on Golden Ratio (1.618) */
--space-1: 0.25rem;   /* 4px */
--space-2: 0.618rem;  /* ~10px */
--space-3: 1rem;      /* 16px */
--space-4: 1.618rem;  /* ~26px */
--space-5: 2.618rem;  /* ~42px */
--space-6: 4.236rem;  /* ~68px */
--space-8: 6.854rem;  /* ~110px */
--space-10: 11.09rem; /* ~177px */

/* Container Widths - Asymmetric */
--container-narrow: 42rem;   /* Blog posts, forms */
--container-reading: 65ch;   /* Optimal reading width */
--container-wide: 90rem;     /* Gallery grids */
--container-full: 100%;      /* Hero sections */
```

### Grid Patterns (Not 12-Column)

```jsx
/* Asymmetric 2-Column Layout (60/40 split) */
<div className="grid grid-cols-[3fr_2fr] gap-space-6">
  <div>{/* Main content */}</div>
  <div>{/* Sidebar */}</div>
</div>

/* Broken Grid Gallery (Masonry-like) */
<div className="grid grid-cols-2 md:grid-cols-3 gap-space-4">
  <div className="row-span-2">{/* Tall image */}</div>
  <div>{/* Square image */}</div>
  <div className="col-span-2">{/* Wide image */}</div>
</div>
```

### Whitespace Rules:
- **Section Padding**: `py-space-10` (not `py-16`)
- **Element Margins**: Use `space-4`, `space-5`, `space-6`
- **Avoid**: `p-4`, `mt-8`, `gap-6`—use custom spacing scale

## Animation Principles - "Slow & Sorcerous"

### Timing Functions (No `ease-in-out`)

```css
/* Custom Cubic Beziers - Organic Motion */
--ease-entrance: cubic-bezier(0.16, 1, 0.3, 1);    /* Slow start, smooth end */
--ease-exit: cubic-bezier(0.7, 0, 0.84, 0);        /* Smooth start, abrupt end */
--ease-mystical: cubic-bezier(0.68, -0.55, 0.27, 1.55); /* Slight overshoot */
```

### Framer Motion Variants

```jsx
// Page transition - Fade in from darkness
const pageVariants = {
  hidden: { opacity: 0 },
  visible: { 
    opacity: 1,
    transition: { duration: 1.2, ease: [0.16, 1, 0.3, 1] }
  }
};

// Scroll-triggered entrance - Elements rise from below
const revealVariants = {
  hidden: { opacity: 0, y: 60 },
  visible: { 
    opacity: 1, 
    y: 0,
    transition: { duration: 0.9, ease: [0.16, 1, 0.3, 1] }
  }
};

// Parallax effect for hero images
const parallaxVariants = {
  initial: { y: 0 },
  scroll: { 
    y: -100,
    transition: { duration: 0 }
  }
};
```

### Animation Rules:
- **All animations**: 500ms-1200ms duration (never 200-300ms like corporate sites)
- **Hover states**: Slow reveals (600-700ms)
- **Scroll animations**: Fade + slide up, triggered at viewport intersection
- **Page transitions**: Full-screen fade (1200ms)

## Accessibility Standards (WCAG 2.1 AA Minimum)

- **Color contrast**: 4.5:1 for body text, 3:1 for large text/UI components
- **Keyboard navigation**: All interactive elements accessible via keyboard
- **Screen readers**: Semantic HTML, ARIA labels where needed
- **Focus indicators**: Clear, visible focus states (2px `--aged-gold` ring)
- **Responsive**: Mobile-first design, min 320px width support
- **Performance**: Lighthouse score >90 on all metrics
- **Alt text**: All images have descriptive alt text (even placeholders)

## What Makes This "Not Corporate"

1. **Warmth in Darkness**: Every black has brown/sepia undertones
2. **Serif Everything**: Even buttons use serif fonts
3. **Asymmetry**: Broken grids, off-center elements
4. **Slow Motion**: All animations 500ms-1200ms
5. **Ornamentation**: SVG flourishes, decorative dividers
6. **Texture Layering**: Noise, vignettes, subtle gradients
7. **Golden Ratio Spacing**: Not rigid 8px grid
8. **Literary Tone**: Grimoire, not sales page

This is a **digital manuscript**, not a SaaS landing page. 📜✨
